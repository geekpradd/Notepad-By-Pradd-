# Notepad by Pradipta Parag (Pradd)


An Open Source Python based Text Editor for Windows XP/Vista/7 and 8.
Version 3.0
Last Updated - 24/10/2014
Coded in PySide(Qt) and Python
Compiled by py2exe. Setup created using InnoSetup.
